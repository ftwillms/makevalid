
makevalid
=========

PostGIS ST_MakeValid equivalent using Shapely

### Installation
```bash
python setup.py install
```

### Run Tests
```bash
python tests/test_validity.py
```

### Sample Usage
Refer to tests/test_validity.py for sample usage.
